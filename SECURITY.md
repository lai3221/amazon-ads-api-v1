# Security policy

## Supported versions

Security fixes are provided for the latest released major version. During initial development, only the latest published release is supported.

## Reporting a vulnerability

Use GitHub private vulnerability reporting for this repository. Do not open a public issue for a suspected vulnerability and do not include Amazon credentials, refresh tokens, access tokens, account identifiers, or customer advertising data in any report.

Include a concise reproduction, affected version, expected impact, and any suggested mitigation. You should receive an initial response within seven days.

For compromised Login with Amazon credentials, rotate or revoke them immediately through the applicable Amazon developer console and contact official Amazon support when necessary.
